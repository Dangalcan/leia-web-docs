# Workbench Frontend

**Repository:** [leia-org/leia-workbench-frontend](https://github.com/leia-org/leia-workbench-frontend)

The web interface used by instructors and students in the Workbench — where experiments are run, sessions are managed, and submissions are reviewed. Built with **React**, **TypeScript**, and **Vite**.

---

## Prerequisites

- **Node.js** >= 18.x
- **npm** or **pnpm**

---

## Local Development

1. Fork and clone the repository:
   ```bash
   git clone <your-fork-url>
   cd leia-workbench-frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:5173`.
